Формула 1 для Homey

Передавайте данные Формулы 1 в реальном времени в свой умный дом Homey.

Приложение подключается к F1 Live Timing (SignalR) и предоставляет状况 трассы,
состояние сессии и данные гонщиков в виде устройств Homey с полной поддержкой
карточек автоматизации.


УСТРОЙСТВА

F1 Трасса
  Статус трассы, погода (температура трассы/воздуха, влажность, ветер),
  сигналы машины безопасности, счётчик кругов

F1 Сессия
  Статус сессии, часы сессии (оставшееся/прошедшее время), сообщения дирекции гонки,
  быстрейший круг

F1 Автомобиль
  Хронометраж на гонщика, состав шин, обнаружение пит-стопов, быстрейший круг


КАРТОЧКИ АВТОМАТИЗАЦИИ

Триггеры (КОГДА)
  - Машина безопасности выехала / вернулась
  - Виртуальная машина безопасности выехала / вернулась
  - Показан красный флаг / Зелёный флаг — трасса свободна
  - Начан новый круг
  - Сессия началась / завершилась
  - Статус сессии изменился
  - Получено сообщение дирекции гонки
  - Изменилась часть квалификации (Q1/Q2/Q3)
  - Быстрейший круг обновлён
  - Гонщик въехал / выехал из пит-лейна
  - Гонщик завершил круг
  - Позиция гонщика изменилась
  - Гонщик установил личный рекорд

Условия (И)
  - Статус трассы — [СВОБОДНО / ЖЁЛТЫЙ / МАШИНА БЕЗОПАСНОСТИ / ВИРТУАЛЬНАЯ МАШИНА БЕЗОПАСНОСТИ / КРАСНЫЙ ФЛАГ]
  - Машина безопасности активна / не активна
  - На трассе идёт / не идёт дождь
  - Сессия активна / не активна
  - Тип сессии — [Гонка / Квалификация / Свободная практика 1/2/3 / Спринт / Квалификация Спринт]
  - Статус сессии — [Неактивна / Начата / Прервана / Завершена / Финализирована / Остановлена]
  - Гонщик в пит-лейне / не в пит-лейне
  - Состав шин — [МЯГКИЙ / СРЕДНИЙ / ЖЁСТКИЙ / ПРОМЕЖУТОЧНЫЙ / ДОЖДЕВОЙ]
  - Позиция гонщика входит / не входит в топ N


ИСТОЧНИК ДАННЫХ

Данные в реальном времени транслируются с livetiming.formula1.com через
классический протокол SignalR. Расписание сезона и результаты берутся из
API Jolpica (совместим с Ergast).


РАЗРАБОТКА

  npm install
  homey app validate
  homey app run

Требуется Homey CLI (npm i -g homey) и устройство Homey в вашей сети.

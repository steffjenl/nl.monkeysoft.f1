Формула 1 для Homey

Транслируйте данные Формулы 1 в реальном времени в свой умный дом Homey.

Приложение подключается к F1 Live Timing (SignalR) и предоставляет условия трассы,
состояние сессии и данные гонщиков в виде устройств Homey с полной поддержкой
карточек автоматизации.


УСТРОЙСТВА

Трасса Ф-1
  Статус трассы, погода (температура трассы/воздуха, влажность, ветер),
  сигналы машины безопасности, счётчик кругов

Сессия Ф-1
  Статус сессии, часы сессии (оставшееся/прошедшее время),
  сообщения дирекции гонки, быстрейший круг

Гонщик Ф-1
  Хронометраж на гонщика, состав шин,
  обнаружение пит-стопов, быстрейший круг


ИСТОЧНИК ДАННЫХ

Данные в реальном времени транслируются с livetiming.formula1.com через классический
протокол SignalR. Расписание сезона и результаты берутся из API Jolpica
(совместим с Ergast).


БЛАГОДАРНОСТИ

Вдохновлено интеграцией F1 Sensor для Home Assistant от Nicxe.
